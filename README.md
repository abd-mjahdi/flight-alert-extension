# SkyAlert

SkyAlert is a browser extension that notifies you when an airplane is flying near your location using live flight data and geolocation.

## Features
- Real-time nearby plane detection  
- Background tracking  
- Instant browser notifications  
- Optional popup UI with flight details  
- Optional live map view  

## Tech Stack
- JavaScript  
- HTML / CSS  
- Browser Extension APIs  
- OpenSky Network API  

## How It Works
1. Gets your location from the browser.  
2. Fetches live flight data.  
3. Calculates distance to nearby aircraft.  
4. Sends a notification when a plane is within range.  

## Setup
1. Clone the repo  
2. Open Chrome → Extensions → Enable Developer Mode  
3. Click "Load unpacked" → Select project folder
